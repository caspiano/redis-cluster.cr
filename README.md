# redis-cluster

redis-cluster library for Crystal

- tested on crystal-0.18.7

## Installation


Add this to your application's `shard.yml`:

```yaml
dependencies:
  redis-cluster:
    github: maiha/redis-cluster.cr
```


## Usage


```crystal
require "redis-cluster"
```
